## Project Overview

- Get nutritional information for each meal
- Display campus-wide options for meals that are
  - High Protein
  - High Carb
  - Vegetarian/Vegan
 
- Filter out superfluous information

Stores data for N amount of time ahead, so that app experience is quicker
